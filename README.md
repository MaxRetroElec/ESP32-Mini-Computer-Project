## MX MEGA-ESP Specifications

Temporary Name : MEGA-ESP

- **CPU** : Xtensa Dual-Core CPU
- **VDP** : Xtensa Dual-Core CPU (with TFT_eSPI library)
- **SOUND CHIP** : Xtensa Dual-Core CPU + MAX98357A (with I2S)
- **KEYBOARD CONTROLLER** : ...............

## Graphics Specifications

Graphics norm for the VDP

- **RESOLUTION** : 320x240
- **PALETTE** : will be used later.

## Description

A mini-computer project made with ESP32, similar to the system of the Commodore PET 2001 or other Personal Computers from the 70-80s
